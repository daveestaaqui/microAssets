# SporlyWorks Design Manifesto: Pure Flat San Francisco Boutique Standard

This document defines the "San Francisco High-End" standard for the SporlyWorks 87+ product portfolio. The DesignAgent must adhere to these specifications for all logo overhauls and UI specifications. **NO 3D, NO DEPTH, NO GRADIENTS.**

## 🎨 Visual Identity Axioms
1. **Geometric Monoline**: All icons must be constructed from mathematically precise, monoline geometric shapes. No variable line weights.
2. **Flagship Motif**: Every product logo must subtly integrate the "Mushroom/Spore" hallmark into its functional geometry. It should feel like a secret signature, not an obvious sticker.
3. **Negative Space**: Leverage "Massive Negative Space." The logo should breathe. 60% of the canvas should be empty to ensure clarity at 16px.
4. **Cream & Charcoal**: 
   - **Charcoal (#1a1a2e)**: The grounding force.
   - **Cream (#faf3e0)**: The organic, premium highlight.
   - *Accent*: Ocean Deep (#0f3460) for interactive states.
5. **Strictly 2D**: No 3D effects, no shadows inside icons, no depth layers. Flat monoline is the absolute mandate for the "Website Format" compatibility.

## 📱 Application & Extension UI
1. **Glassmorphism**: Use translucent panels with subtle blurs for a futuristic, premium feel.
2. **Typography (The "Inter" Rule)**: Use Inter or Outfit. No serifs. Tight tracking for headers, generous leading for body text.
3. **Micro-Transitions**: All buttons and hover states must have a 200ms ease-in-out transition. Static interfaces are prohibited.
4. **Shadow Hierarchy**: Use layered, soft "dreamy" shadows (box-shadow: 0 10px 30px rgba(0,0,0,0.1)) instead of harsh borders.

## 🛡️ Quality Enforcement
- **Asset Integrity**: Any icon that pixelates at 500% zoom is a failure. Use SVG exclusively.
- **Brand Consistency**: If the 'Notes' extension doesn't look like it belongs to the same family as the 'Stock Tracker,' the overhaul is incomplete.
